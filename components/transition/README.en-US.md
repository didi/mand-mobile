### API
`md-transition` is a wrapper of vue `transtion`, support all `transition` props.

built-in transition name：

- `md-fade`
- `md-fade-up`
- `md-fade-down`
- `md-fade-left`
- `md-fade-right`
- `md-slide-up`
- `md-slide-down`
- `md-slide-left`
- `md-slide-right`
- `md-bounce`
- `md-punch`
- `md-zoom`
