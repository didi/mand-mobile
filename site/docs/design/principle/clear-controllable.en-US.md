title: Clear and controllable
---

The user should be given the right to know and independent control. The user can clearly know the results of the action to be carried out, and can understand the current position and free return in operation.

#### Information transparency

The financial scenario involves the signing of various agreements. In order to facilitate users' understanding, the display of the agreement should be clear and simple, and users' active authorization is required.

When the user is carrying out complex tasks, try to inform the user of the current task progress and the remaining tasks to be completed.

#### Let users decide

For operations that affect the user's attention or transaction, the user shall be allowed to revoke / modify the results generated by the operation, and the user shall be given the right to operate at both the source and the end.
